# NOvA Analysis Showcase

This repository is intended as a public code showcase. It contains selected analysis scripts from my atmospheric neutrino research, but experiment-specific input files, private datasets, and collaboration infrastructure are not included.

## Structure
- `analysis/`: event-level data preparation and spectrum-building scripts
- `validation/`: reconstruction validation and directional-resolution studies
- `inference/`: statistical inference and fitting workflows

## Notes
- This is a curated public-facing showcase repository.
- Large detector files, experiment-specific datasets, and generated outputs are not included.
- Some paths and dependencies may need to be adapted for external reuse.

## Skills demonstrated
- scientific computing in C++
- ROOT-based data analysis
- event-level data pipelines
- reconstruction validation
- statistical inference
- large-scale analysis workflow design
